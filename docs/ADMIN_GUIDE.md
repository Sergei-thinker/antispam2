## Начало работы

После настройки бота (см. [Инструкцию по установке](SETUP.md)) все команды доступны только **администраторам** — пользователям, чьи Telegram ID указаны в переменной `ADMIN_IDS`.

Для взаимодействия с ботом отправляйте команды в **личный чат** с ботом (не в групповой чат).

### Проверка работоспособности

Отправьте боту команду `/status`. Если бот работает корректно, вы получите ответ:

```
🟢 Статус бота: Активен

📊 Информация:
• Модель ИИ: anthropic/claude-sonnet-4
• Порог уверенности: 0.70
• Время работы: 2д 14ч 32м
• Лимит запросов: 15/20 в минуту
• Размер БД: 2.4 МБ
• Канал: -1001234567890
```

## Команды бота

### `/start` — Приветствие

Отправляет приветственное сообщение с кратким описанием бота.

**Пример:**
```
Вы: /start

Бот: 👋 Добро пожаловать в Антиспамер 2!

Я отслеживаю комментарии в вашем канале и автоматически
удаляю спам с помощью ИИ.

Используйте /help для списка команд.
```

---

### `/help` — Список команд

Показывает все доступные команды с описаниями.

**Пример:**
```
Вы: /help

Бот: 📋 Доступные команды:

/start — Приветствие
/help — Список команд (это сообщение)
/stats — Статистика работы бота
/status — Текущий статус бота
/whitelist <user_id> — Добавить пользователя в белый список
/unban <user_id> — Разбанить пользователя
/recent — Последние обнаружения спама
```

---

### `/whitelist <user_id>` — Белый список

Добавляет пользователя в белый список. Сообщения пользователей из белого списка **никогда** не проверяются на спам.

**Пример:**
```
Вы: /whitelist 123456789

Бот: ✅ Пользователь 123456789 добавлен в белый список.
Его сообщения больше не будут проверяться на спам.
```

**Ошибки:**
```
Вы: /whitelist

Бот: ❌ Укажите ID пользователя.
Использование: /whitelist <user_id>
```

```
Вы: /whitelist abc

Бот: ❌ Некорректный ID пользователя. ID должен быть числом.
```

```
Вы: /whitelist 123456789
(если уже в списке)

Бот: ℹ️ Пользователь 123456789 уже находится в белом списке.
```

---

### `/unban <user_id>` — Разбанить

Снимает бан с пользователя и удаляет его из списка забаненных. Пользователь снова сможет писать комментарии в группе обсуждения.

**Пример:**
```
Вы: /unban 123456789

Бот: ✅ Пользователь 123456789 разбанен.
Он снова может писать комментарии в группе.
```

**Ошибки:**
```
Вы: /unban 123456789
(если не был забанен)

Бот: ℹ️ Пользователь 123456789 не найден в списке забаненных.
```

**Примечание:** Разбанить пользователя также можно через кнопку в уведомлении о спаме (см. раздел ниже).

---

## Уведомления о спаме

Когда бот обнаруживает и удаляет спам, все администраторы получают уведомление в личный чат с ботом.

### Формат уведомления

```
🚫 Спам обнаружен и удалён!

👤 Пользователь: CryptoKing2024
🆔 ID: 987654321
📛 Username: @cryptoking2024

💬 Сообщение:
"Заработай $5000 за неделю без усилий! Переходи
на наш сайт и начни зарабатывать прямо сейчас!
100% гарантия. Напиши мне в ЛС для подробностей."

🎯 Уверенность ИИ: 0.97 (97%)
📝 Причина: Мошенническая реклама заработка,
типичный паттерн финансового скама

⚡ Действия:
• Сообщение удалено ✅
• Пользователь забанен ✅

[🔓 Разбанить пользователя]   ← кнопка
```

### Кнопка «Разбанить пользователя»

Каждое уведомление содержит инлайн-кнопку для быстрого разбана. Это полезно, если бот ошибся (ложное срабатывание).

При нажатии кнопки:
1. Пользователь разбанивается в группе обсуждения
2. Запись о бане помечается как отменённая в базе данных
3. Сообщение обновляется:

```
🚫 Спам обнаружен и удалён!
...
(то же содержание)
...

⚡ Действия:
• Сообщение удалено ✅
• Пользователь разбанен ↩️ (разбанил: @admin_username)
```

## Управление банами

### Автоматические баны

Бот автоматически банит пользователей, чьи сообщения классифицированы как спам с уверенностью **выше порога** (по умолчанию 0.70 = 70%).

**Что происходит при бане:**
1. Сообщение удаляется из группы обсуждения
2. Пользователь банится (не может писать новые сообщения)
3. Информация сохраняется в базу данных
4. Администраторы получают уведомление

### Разбан через команду

```
/unban <user_id>
```

Используйте ID пользователя из уведомления о спаме или из команды `/recent`.

### Разбан через кнопку

Нажмите кнопку «Разбанить пользователя» в уведомлении о спаме. Это самый быстрый способ отменить ошибочный бан.

### Просмотр забаненных

Команда `/status` показывает общее количество забаненных пользователей. Для детального списка используйте `/recent` чтобы увидеть последние баны.

## Белый список

Белый список — это список пользователей, чьи сообщения **никогда** не проверяются на спам. Используйте его для:

- **Активных комментаторов** — постоянные подписчики, которых бот иногда ошибочно блокирует
- **Модераторов** — пользователи, которые помогают с модерацией
- **Партнёров** — аккаунты, которым разрешена реклама

### Добавление в белый список

```
/whitelist 123456789
```

### Удаление из белого списка

На данный момент удаление из белого списка выполняется через базу данных. Планируется добавление команды `/unwhitelist` в будущих версиях.

### Кто автоматически пропускается

Помимо белого списка, бот автоматически пропускает:
- **Ботов** — все сообщения от ботов игнорируются
- **Администраторов** — пользователи из `ADMIN_IDS` всегда пропускаются

## Статистика

### Чтение статистики

Команда `/stats` показывает:

| Метрика | Описание |
|---------|----------|
| Сообщений обработано | Общее количество проверенных сообщений |
| Спам обнаружен | Сообщения, классифицированные как спам |
| Пользователей забанено | Уникальных пользователей забанено |
| Ложных срабатываний | Разбаненные пользователи (предполагаемые ошибки) |
| Точность | (1 - ложные_срабатывания / спам_обнаружен) × 100% |
| Вызовы ИИ | Количество обращений к API OpenRouter |
| Ошибки ИИ | Неудачные обращения к API |
| Средняя уверенность | Средний показатель уверенности ИИ для спама |

### Интерпретация статистики

**Хорошие показатели:**
- Точность > 95%
- Ошибки ИИ < 2%
- Средняя уверенность > 0.85

**Требуют внимания:**
- Точность < 90% → Слишком много ложных срабатываний. Увеличьте `SPAM_CONFIDENCE_THRESHOLD`
- Ошибки ИИ > 5% → Проблемы с OpenRouter API. Проверьте ключ и баланс
- Много пропущенных (лимит) → Увеличьте `MAX_AI_CALLS_PER_MINUTE` или смените модель на более быструю

## Часто задаваемые вопросы

### Бот не отвечает на команды

**Возможные причины:**
1. Бот не запущен — проверьте серверный процесс
2. Вы не в списке администраторов — проверьте `ADMIN_IDS` в `.env`
3. Вы пишете в группу, а не в личный чат с ботом

**Решение:**
- Убедитесь, что бот запущен (`python main.py`)
- Проверьте, что ваш Telegram ID есть в `ADMIN_IDS`
- Отправьте команду в **личный чат** с ботом

---

### Бот не видит комментарии в канале

**Возможные причины:**
1. Бот не добавлен в группу обсуждения как администратор
2. Включён режим приватности (Group Privacy)
3. Неправильный `CHANNEL_ID`

**Решение:**
- Добавьте бота как администратора в **группу обсуждения** (не в сам канал)
- В BotFather: `/mybots` → ваш бот → Bot Settings → Group Privacy → выключить
- Проверьте `CHANNEL_ID` в `.env`

---

### Слишком много ложных срабатываний

Бот блокирует нормальные комментарии.

**Решение:**
1. Увеличьте порог уверенности:
   ```env
   SPAM_CONFIDENCE_THRESHOLD=0.85
   ```
2. Добавьте частых комментаторов в белый список:
   ```
   /whitelist <user_id>
   ```
3. Используйте более точную модель ИИ:
   ```env
   AI_MODEL=anthropic/claude-sonnet-4
   ```
4. Перезапустите бота после изменения `.env`

---

### Бот пропускает спам

Спам проходит через фильтр.

**Решение:**
1. Уменьшите порог уверенности:
   ```env
   SPAM_CONFIDENCE_THRESHOLD=0.5
   ```
2. Убедитесь, что API-ключ OpenRouter работает и есть баланс
3. Проверьте логи бота на наличие ошибок API
4. Используйте более мощную модель:
   ```env
   AI_MODEL=anthropic/claude-opus-4
   ```

### Как узнать ID пользователя?

1. Перешлите любое сообщение пользователя боту [@userinfobot](https://t.me/userinfobot)
2. Бот ответит с ID пользователя
3. Также ID показывается в уведомлениях о спаме и в команде `/recent`

---

### Как изменить модель ИИ?

1. Откройте файл `.env`
2. Измените значение `AI_MODEL`:
   ```env
   AI_MODEL=anthropic/claude-haiku-4-5
   ```
3. Перезапустите бота

Список доступных моделей: [openrouter.ai/models](https://openrouter.ai/models)

---

### Бот удалил сообщение по ошибке, что делать?

1. Нажмите кнопку **«Разбанить пользователя»** в уведомлении
2. Или используйте команду `/unban <user_id>`
3. К сожалению, удалённое сообщение восстановить нельзя
4. Рассмотрите возможность добавить пользователя в белый список, чтобы предотвратить повторные ошибки

---

### Сколько стоит использование бота?

Бот бесплатный, но использование ИИ через OpenRouter стоит денег:
- Для канала с ~50 комментариями в день: **$1–6 в месяц** (зависит от модели)
- Для канала с ~100 комментариями в день: **$2–12 в месяц**
- См. подробный расчёт в [документации по API](API.md)

### Как обновить бота?

```bash
cd antispamer-2
git pull
pip install -r requirements.txt
# Перезапустите бота
```

База данных сохраняется при обновлении — все баны, белый список и статистика останутся на месте.
